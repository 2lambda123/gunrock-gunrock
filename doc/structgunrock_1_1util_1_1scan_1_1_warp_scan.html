Detailed Description

template<int LOG_NUM_ELEMENTS, bool EXCLUSIVE = true, int STEPS = LOG_NUM_ELEMENTS>
struct gunrock::util::scan::WarpScan< LOG_NUM_ELEMENTS, EXCLUSIVE, STEPS >

Performs STEPS steps of a Kogge-Stone style prefix scan.

Requires a 2D "warpscan" structure of smem storage having dimensions [2][NUM_ELEMENTS].
