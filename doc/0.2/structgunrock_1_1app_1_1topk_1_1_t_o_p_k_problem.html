Detailed Description

template<typename _VertexId, typename _SizeT, typename _Value>
struct gunrock::app::topk::TOPKProblem< _VertexId, _SizeT, _Value >

TOPK Problem structure stores device-side vectors for doing TOPK on the GPU.

Template Parameters
_VertexIdType of signed integer to use as vertex id (e.g., uint32)
_SizeTType of unsigned integer to use for array indexing. (e.g., uint32)
_ValueType of float or double to use for computing degree centrality value.
