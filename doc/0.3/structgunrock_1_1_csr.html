Detailed Description

template<typename VertexId, typename Value, typename SizeT>
struct gunrock::Csr< VertexId, Value, SizeT >

CSR data structure which uses Compressed Sparse Row format to store a graph. It is a compressed way to present the graph as a sparse matrix.
