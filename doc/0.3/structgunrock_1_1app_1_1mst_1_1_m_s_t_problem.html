Detailed Description

template<typename VertexId, typename SizeT, typename Value, bool _MARK_PREDECESSORS, bool _ENABLE_IDEMPOTENCE, bool _USE_DOUBLE_BUFFER>
struct gunrock::app::mst::MSTProblem< VertexId, SizeT, Value, _MARK_PREDECESSORS, _ENABLE_IDEMPOTENCE, _USE_DOUBLE_BUFFER >

MST Problem structure stores device-side vectors for doing Minimum Spanning Tree on the GPU.

Template Parameters
_VertexIdType of signed integer to use as vertex id (e.g., uint32)
_SizeTType of unsigned integer to use for array indexing. (e.g., uint32)
_ValueType of float or double to use for computing MST value.
_MARK_PREDECESSORSWhether to mark predecessor value for each node.
_ENABLE_IDEMPOTENCEWhether to enable idempotent operation.
_USE_DOUBLE_BUFFERDefines whether to use double buffer.
