Reads a MARKET graph from an input-stream into a CSR sparse format.

Here is an example of the matrix market format +-------------------------------------------—+ |%MatrixMarket matrix coordinate real general | <— header line |% | <–+ |% comments | |– 0 or more comment lines |% | <–+ | M N L | <— rows, columns, entries | I1 J1 A(I1, J1) | <–+ | I2 J2 A(I2, J2) | | | I3 J3 A(I3, J3) | |– L lines | . . . | | | IL JL A(IL, JL) | <–+ +-------------------------------------------—+

Indices are 1-based i.2. A(1,1) is the first element.

Parameters
[in]f_inInput MARKET graph file.
[in]output_fileOutput file name for binary i/o.
[in]csr_graphCsr graph object to store the graph data.
[in]undirectedIs the graph undirected or not?
[in]reversedWhether or not the graph is inversed.
Returns
If there is any File I/O error along the way.
