Detailed Description

template<typename VertexId, typename SizeT, typename Value, bool _MARK_PREDECESSORS, bool _ENABLE_IDEMPOTENCE, bool _USE_DOUBLE_BUFFER>
struct gunrock::app::bfs::BFSProblem< VertexId, SizeT, Value, _MARK_PREDECESSORS, _ENABLE_IDEMPOTENCE, _USE_DOUBLE_BUFFER >

Breadth-First Search Problem structure stores device-side vectors for doing BFS computing on the GPU.

Template Parameters:
VertexIdType of signed integer to use as vertex id (e.g., uint32)
SizeTType of unsigned integer to use for array indexing. (e.g., uint32)
ValueType of float or double to use for computing BC value.
_MARK_PREDECESSORSBoolean type parameter which defines whether to mark predecessor value for each node.
_ENABLE_IDEMPOTENCEBoolean type parameter which defines whether to enable idempotence operation for graph traverse.
_USE_DOUBLE_BUFFERBoolean type parameter which defines whether to use double buffer.
