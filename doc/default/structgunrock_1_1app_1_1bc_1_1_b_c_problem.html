template<typename VertexId, typename SizeT, typename Value, bool _MARK_PREDECESSORS, bool _USE_DOUBLE_BUFFER>
struct gunrock::app::bc::BCProblem< VertexId, SizeT, Value, _MARK_PREDECESSORS, _USE_DOUBLE_BUFFER >

Betweenness centrality problem data structure which stores device-side vectors for doing BC computing on the GPU.

Template Parameters:
_VertexIdType of signed integer to use as vertex id (e.g., uint32)
_SizeTType of unsigned integer to use for array indexing. (e.g., uint32)
_ValueType of float or double to use for computing BC value.
_MARK_PREDECESSORSWhether to mark predecessor value for each node.
_USE_DOUBLE_BUFFERBoolean type parameter which defines whether to use double buffer.
